Доска задач для новичков:

Чек-лист первого контрибьюта:
1. Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2. Клонируйте свой форк локально:
   git clone https://github.com/ВАШ ЛОГИН/Ваше имя-mgutu-fct.git
3. Перейдите в папку проекта:
   сd ваше имя-mgutu-fct
4. Установите зависимости:
   pip install -r requirements.txt
5. Запустите тесты:
   python -m pytest tests/
6. Внесите правки, зафиксируйте, отправьте PR.
